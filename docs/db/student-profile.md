# 生徒カルテDB

> 生徒ごとの成長を一目で把握する。

## カラム定義

| # | カラム名 | 型 | 選択肢/設定 | 説明 |
|---|---------|------|-----------|------|
| 1 | 生徒名 | Title | - | |
| 2 | 学年 | Select | 小5 / 小6 / 中1 / 中2 | |
| 3 | 開始日 | Date | - | スクール開始日 |
| 4 | 現在のレッスン | Select | Lesson 1〜Lesson 18 | 今どこまで進んでいるか |
| 5 | 授業ログ | Relation | → 授業ログDB | 過去の全レッスンログ |
| 6 | 累計レッスン数 | Rollup | 授業ログの件数をカウント | 何回授業したか |
| 7 | 自力解決率 | Rollup/Formula | 授業ログの「自力で解決」の割合 | トライ&エラー力の指標 |
| 8 | 得意なこと | Text | - | |
| 9 | 苦手・注意点 | Text | - | |
| 10 | 性格メモ | Text | - | 「負けず嫌い」「慎重派」など |
| 11 | 保護者メモ | Text | - | 保護者から聞いた情報 |
| 12 | ステータス | Select | 受講中 / 休会中 / 卒業 | |

## ビュー

| ビュー名 | 種類 | 用途 |
|---------|------|------|
| 受講中の生徒 | Table (フィルタ: 受講中) | メインビュー |
| カード形式 | Gallery | 生徒一覧を見やすく |

## 使い方

- 新しい生徒が入ったらレコード追加
- 授業ログDB と Relation で接続 → 過去の全記録が紐づく
- 「自力解決率」で、トライ&エラーの自走度をざっくり把握
