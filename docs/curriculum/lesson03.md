## 生徒用チャレンジ

### プログラミングって何？（5分）

**講師が対話形式で**:

「友達に何かお願いするとき、どうする？」→「言葉で伝えるよね」

「コンピュータも同じ。でもコンピュータ用の言葉がある。Roblox では **Luau（ルアウ）** っていう言葉を使うよ」

**たとえ話**:
- プログラミング = コンピュータへの「お手紙」
- スクリプト = お手紙を入れる「封筒」
- Play = お手紙を「届ける」

---

### チャレンジ 1: Hello World!（15分）

**ミッション**: 「コンピュータに最初のメッセージを送ろう！」

**手順**:
1. ブロックを1個作る
2. Explorer でブロックの ＋ → Script を追加
3. スクリプトをダブルクリックして開く
4. 最初の文字を全部消して、以下を書く:
5. **View → Output** を開く（超重要！先に開く）
6. Play を押す

```lua
print("Hello World!")
```

**ヒント**:
- Lv.1: 「`print` って打ってみて。"表示して！"っていう命令だよ」
- Lv.2: 「カッコの中に `"` で囲んだ文字を入れるよ」
- Lv.3: 一緒に1文字ずつ打つ

**成功した瞬間**: 「Output に Hello World! って出た？これ、世界中のプログラマーが最初に書くやつだよ。君もプログラマーの仲間入り！」

---

### チャレンジ 2: いろんなものを表示しよう（10分）

**ミッション**: 「print でいろんなものを表示させてみよう！」

```lua
-- 自分の名前
print("ぼくの名前は太郎です")

-- 計算もできる！
print(1 + 1)
print(100 - 37)
print(3 * 4)

-- 文字と数字を組み合わせる
print("答えは " .. 2 + 3)
```

**ポイント**:
- `--` で始まる行はコメント（コンピュータは無視する。自分用メモ）
- print の中に計算式を書くと答えが出る
- エラーが出たらチャンス！「コンピュータからのお手紙を読んでみよう」

---

### チャレンジ 3: 変数 — 名前をつけて覚えさせよう（10分）

**ミッション**: 「コンピュータに"覚えて"って言ってみよう」

```lua
local name = "太郎"
print(name)
print("こんにちは、" .. name .. "さん！")
```

**たとえ話**: 「変数 = "名前のついた箱"。`name` って箱に "太郎" を入れる。箱の名前を言えば中身が出てくる」

**自分のプロフィールを作ろう**:

```lua
local name = "（自分の名前）"
local age = （自分の年齢）
local hobby = "（好きなこと）"
print(name .. "、" .. age .. "歳、趣味は" .. hobby)
```

**よくあるつまづき**:
| エラー | 原因 | 声かけ |
|--------|------|--------|
| `local` が抜けてる | 書き忘れ | 「最初に local って書くのがルール」 |
| `"` で囲み忘れ | 文字列ルール | 「文字は必ずダブルクォーテーションで囲む」 |
| `..` がわからない | 初見 | 「点を2つ（..）で文字をくっつけるよ」 |

---

### チャレンジ 4: ブロックに触ったらメッセージ！（15分）

**ミッション**: 「ブロックに触ったら Output に "タッチ！" って出るようにしよう」

```lua
local part = script.Parent

part.Touched:Connect(function()
    print("タッチ！")
end)
```

**1行ずつ説明**:
| コード | 意味 |
|--------|------|
| `local part = script.Parent` | このスクリプトが入ってるブロックを `part` と呼ぶ |
| `part.Touched:Connect(function()` | `part` に何かが触ったら... |
| `print("タッチ！")` | "タッチ！"と表示する |
| `end)` | おわり |

**テスト**: Play → キャラクターをブロックに歩かせる → Output に「タッチ！」が出る！

---

### チャレンジ 5: メッセージを変えてみよう（5分）

**ミッション**: 「print の中身を変えたり、変数を使ったりしてみよう」

```lua
local part = script.Parent
local message = "ブロックに触ったよ！"

part.Touched:Connect(function()
    print(message)
end)
```

自由に試す:
- `print("いたっ！")` / `print("見つけた！")` / `print("ここだよ！")`

## 講師ガイド

### この回の核心
「print と変数と Touched を覚えること」ではなく、**「自分が書いた文字がコンピュータに伝わった！」という感動体験**。最初の一歩を「楽しい！もっとやりたい！」で終わらせる。

### 授業前チェック（3分）
- [ ] 前回の授業ログ確認
- [ ] Output ウィンドウの出し方を把握（View → Output）

### 超重要: Output を先に開かせる
print を書いても Output が見えないと「何も起きない」ように見える。**チャレンジ 1 の前に必ず View → Output**。

### エラーとの付き合い方（ここで文化を作る）
最初のエラーが出た瞬間が最大のチャンス。

- 「やった！エラーだ！コンピュータが"ここ違うよ"って教えてくれてるよ」
- 絶対に「間違ってるよ」と言わない
- 「コンピュータからのお手紙を読んでみよう」

### 次回への接続
振り返りで「次回から"コイン集めアドベンチャー"を作る」と予告。「今日の print と Touched は、次回以降めちゃくちゃ使うよ！」
