### チャレンジ 2: TweenService で動かそう（20分）

**ミッション**: 「コードを書いて、ブロックを左右に動かしてみよう」

**導入**: 「今までは Touched で"触ったら何かする"をやったよね。今日は TweenService っていう"動かす道具"を使うよ」

**書くコード**:

```lua
local TweenService = game:GetService("TweenService")
local part = script.Parent

-- 動きの設定
local tweenInfo = TweenInfo.new(
    2,                          -- 2秒かけて動く
    Enum.EasingStyle.Linear,    -- 一定の速度
    Enum.EasingDirection.InOut,
    -1,                         -- 無限に繰り返す
    true                        -- 行ったり来たりする
)

-- 目的地（今の位置から横に10移動）
local goal = {Position = part.Position + Vector3.new(10, 0, 0)}

-- 動かす！
local tween = TweenService:Create(part, tweenInfo, goal)
tween:Play()
```

**1行ずつ説明（全部は説明しない。重要な部分だけ）**:

| コード | 意味 |
|--------|------|
| `game:GetService("TweenService")` | 「動かす道具」を取り出す |
| `TweenInfo.new(2, ...)` | 2秒かけて動く設定 |
| `-1` | 無限に繰り返す |
| `true` | 行ったり来たりする |
| `Vector3.new(10, 0, 0)` | 横に10移動（X方向） |
| `tween:Play()` | 動かす！ |

**たとえ話**: 「TweenService は"ロボットアーム"みたいなもの。"このブロックを、こっちに、2秒かけて、行ったり来たりで動かして"って指示するんだ」

**想定つまづき（講師用）**:
- コードが長くて怖い → 「全部理解しなくて大丈夫。変えるのは数字だけだよ」
- 動かない → Anchored がオフかも / スクリプトの場所を確認
- 変な方向に動く → Vector3 の数字を調整

### チャレンジ 3: 数字を変えて遊ぼう（10分）

**ミッション**: 「数字を変えると動き方が変わるよ。いろいろ試してみて！」

**変えられる数字**:
- `2` → 速さ（1 = 速い、5 = ゆっくり）
- `Vector3.new(10, 0, 0)` → 方向と距離
  - `(10, 0, 0)` = 横
  - `(0, 10, 0)` = 上下
  - `(0, 0, 10)` = 奥行き
  - `(10, 5, 0)` = 斜め！

**自由に試す**: 上下に動く床、斜めに動く床、ゆっくり動く床など

### チャレンジ 5: 発展 — 開閉する扉（15分）

**ミッション**: 「扉が上に動いて道が開く仕掛けを作ってみよう」

```lua
local TweenService = game:GetService("TweenService")
local door = script.Parent

local openInfo = TweenInfo.new(1, Enum.EasingStyle.Quad)
local openGoal = {Position = door.Position + Vector3.new(0, 8, 0)}

local function openDoor()
    local tween = TweenService:Create(door, openInfo, openGoal)
    tween:Play()
end

door.Touched:Connect(function(hit)
    local humanoid = hit.Parent:FindFirstChild("Humanoid")
    if humanoid then
        openDoor()
    end
end)
```

「触ったら上に動いて道が開く！ Touched と TweenService の組み合わせだよ」

**発展アイデア**: 「一定数のコインを集めたら開く扉」は次回やってもOK
