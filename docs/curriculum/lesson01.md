## 生徒用チャレンジ

### チャレンジ 1: ブロックを出そう（5分）

**ミッション**: 「灰色の地面の上に、ブロックを1個出してみて！」

**ヒント**:
- Lv.1: 「画面の一番上に "Home" っていうタブがあるよ。見つかる？」
- Lv.2: 「Home の中に "Part" って書いてあるボタンがあるよ」
- Lv.3: 「Home → Part → Block を押してみて」

> 公式: [Parts](https://create.roblox.com/docs/parts)

---

### チャレンジ 2: ブロックを動かそう（5分）

**ミッション**: 「ブロックを好きな場所に動かせる？」

**ヒント**:
- Lv.1: 「ブロックを選ぶと矢印が出るよ。引っ張ってみて」
- Lv.2: 「赤が横、緑が上下、青が奥行き」
- Lv.3: 「矢印の"先っぽ"を掴んでね。本体じゃないよ」

**よくあるつまづき**: パーツ本体をドラッグしてしまう → 「矢印の先っぽを掴んでね」

**ヒント**:

---

### チャレンジ 4: 好きな色をつけよう（5分）

**ミッション**: 「ブロックに好きな色をつけてみて！」

**ヒント**:
- Lv.1: 「ブロックを選んで、右のパネルを見てみて」
- Lv.2: 「Properties の中に色を変える場所があるよ」
- Lv.3: 「BrickColor の横のボックスをクリックしてみて」

---

### チャレンジ 5: コピーしてたくさん作ろう（5分）

**ミッション**: 「同じブロックを一瞬でコピーする方法を見つけて！」

**ヒント**:
- Lv.1: 「パソコンで"コピー"するとき、どうする？似た方法だよ」
- Lv.2: 「Ctrl（Mac: Cmd）を使うよ」
- Lv.3: 「Ctrl+D（Mac: Cmd+D）で複製！」

**ここで教える超重要ショートカット**:

| ショートカット | Mac | 機能 | なぜ重要か |
|--------------|-----|------|-----------|
| Ctrl+D | Cmd+D | 複製 | 作業効率化 |
| Ctrl+Z | Cmd+Z | 元に戻す | **失敗を恐れずに試せる安全網** |
| Ctrl+S | Cmd+S | 保存 | データを守る |

> Ctrl+Z は何よりも先に教える。「失敗しても戻せるよ」がトライ&エラーの土台。

---

### チャレンジ 6: 自分だけの部屋を作ろう！（35分）

**ミッション**: 「覚えた技を全部使って、自分だけの部屋を作ろう！壁、床、家具、なんでもOK！」

**Anchored について（Play して初めて教える）**:

生徒が Play ボタンを押す → ブロックが重力で落下 → パニック。

「落ちちゃったね！Ctrl+Z で戻そう。落ちないようにする方法を探してみて」→ Properties → Anchored → チェック。

**これが Lesson 1 最高のトライ&エラー体験**: 問題発生 → 原因を探す → 自分で解決する。

## 講師ガイド

### この回の核心
「部屋を作ること」ではなく、**Studio で「自分で触って発見する」体験**をさせること。

### 授業前チェック（3分）
- [ ] 生徒の PC に Roblox Studio インストール済み
- [ ] アカウント作成済み（13歳未満は保護者同意）
- [ ] Mac/Windows の確認（ショートカットが異なる）

### カメラ操作（最初に教える）
| 操作 | 動き |
|------|------|
| 右クリック+ドラッグ | 視点回転 |
| スクロール | ズームイン/アウト |
| 中クリック+ドラッグ | パン（平行移動） |

### 自由制作中の対応表
| 生徒の状態 | 対応 |
|-----------|------|
| 手が止まっている | 「何を作りたい？」→ アイデアを一緒に出す |
| 同じことを繰り返す | 「他にもやり方あるかも？」 |
| 夢中で作っている | **見守る。邪魔しない。** |
| Play したら落ちた | 「なんで落ちたと思う？」→ Anchored の発見へ |
| 巨大パーツで遊んでいる | 止めない。問題が起きてから教える |

### NG / OK
| NG | OK |
|----|----|
| 全手順を最初に言う | 「まず触ってみて。何か見つかった？」 |
| 生徒のマウスを奪って操作する | 「何がしたい？」と聞く |
| 「違う、そうじゃなくて」 | 「面白い！なんでそうなったか調べよう」 |
