# 講師の1週間フロー

## 授業前（3分）

1. **レッスンページを開く** → 「講師メモ」セクションを読む
2. **前回の授業ログを確認** → この生徒は前回何でつまづいた？どんな反応だった？
3. **心構え**: 今日は「教える」ではなく「観察して問いかける」

## 授業中（80分）

### 基本サイクル

```
チャレンジを出す（ミッションを読む）
    ↓
生徒が自分で試す（見守る）
    ↓
つまづく
    ↓
ヒントを段階的に出す（Lv.1 → Lv.2 → Lv.3）
    ↓
解決 or 講師が介入
    ↓
「なんでうまくいった（いかなかった）と思う？」
    ↓
次のチャレンジへ
```

### 介入のタイミング

| 状態 | 待つ時間 | アクション |
|------|---------|-----------|
| 手が動いている | 無限 | 見守る |
| 手が止まった | 1分 | 「今なに考えてる？」と聞く |
| 困っている顔 | 30秒 | ヒント Lv.1 を出す |
| フリーズ（完全に止まった） | すぐ | 「何がしたい？」から一緒に整理 |

### 声かけのフレーズ集

**やってはいけない（NG）**:
- 「違う違う、そうじゃなくて」
- 「これをこうして、次にこうして...」（手順を全部言う）
- マウスを奪って操作する

**使ってほしい（OK）**:
- 「まず触ってみて。何か見つかった？」
- 「うまくいかなかったね。なんでだと思う？」
- 「いい発見だね！それ使えそう？」
- 「面白い！なんでそうなったか調べてみよう」
- 「どうやったの？」（成功した後に言語化させる）

## 授業後（5分）

### 授業ログDBに1レコード追加

記録する内容:
1. **つまづいた箇所** — 何で詰まったか
2. **生徒の反応** — 自力解決？ヒント？講師介入？
3. **トライ&エラー度** — 自分から試す姿勢（★〜★★★）
4. **振り返りの回答** — 生徒の言葉をそのまま記録
5. **気づいたこと** — 次回への引き継ぎメモ
6. **教材改善メモ** — 「ここの説明変えた方がいい」があれば

### Phase ごとの講師スタンス変化

| Phase | 講師の介入レベル | 心がけ |
|-------|--------------|--------|
| **Phase 1**（Lesson 1-6） | 高め | ヒントは多めに。「楽しい」が最優先 |
| **Phase 2**（Lesson 7-14） | 中くらい | ヒントを減らす。「公式ドキュメント見てみて」を促す |
| **Phase 3**（Lesson 15-18） | 最小限 | 生徒が主導。講師は質問に答えるだけ |
