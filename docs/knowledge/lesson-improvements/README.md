# レッスンプラン改善

授業ログから抽出した、教材やレッスンの流れに対する改善提案。

## インデックス

- [Lesson 1 改善提案](knowledge/lesson-improvements/lesson01.md)

## 優先度の定義

| 優先度 | 基準 |
|--------|------|
| **高** | 複数の生徒で同じ問題が発生 / 授業進行に支障 |
| **中** | 改善すればスムーズになるが、なくても進行可能 |
| **低** | あると良いが、急がなくてよい |

## 改善の反映フロー

```
改善提案を蓄積
    ↓ 3〜5件溜まったらレビュー
カリキュラム（/curriculum）を修正
    ↓
git push → サイト反映
```
