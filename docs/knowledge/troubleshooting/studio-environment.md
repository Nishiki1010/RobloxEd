# Roblox Studio 環境系トラブル

---

### Studio の言語が英語のまま

- **症状**: メニューやプロパティがすべて英語で表示される
- **原因**: デフォルト設定が英語
- **解決**: Studio Settings → General → Language → 日本語
- **補足**: 教材は英語UI前提で作成しているため、英語のままでも問題ない。生徒の好みに合わせる
- **発生レッスン**: Lesson 1
- **元データ**: [2026-03-09_sample_L01](logs/2026-03-09_sample_L01.md)

---

### Properties ウィンドウが表示されない

- **症状**: ブロックを選択しても右側にプロパティが表示されない
- **原因**: ウィンドウが閉じられている
- **解決**: View タブ → Properties にチェック
- **予防**: 授業開始時に View → Properties / Explorer / Output が表示されていることを確認
- **発生レッスン**: Lesson 1, 3

---

### Output ウィンドウが表示されない

- **症状**: スクリプトエラーが確認できない
- **原因**: ウィンドウが閉じられている
- **解決**: View タブ → Output にチェック
- **予防**: Lesson 3（初スクリプト）の前に必ず開かせる
- **発生レッスン**: Lesson 3
