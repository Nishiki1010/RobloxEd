## 教育哲学

子供たちが磨くべきは **Robloxの知識そのもの** ではなく：

1. **仮説を立てる** — 「こうしたらどうなるかな？」
2. **実行する** — 実際にやってみる
3. **エラーから学ぶ** — 「なんで動かなかったんだろう？」
4. **やり直す** — 修正して再挑戦する

このフィードバックループを自走できる子供を育てます。
